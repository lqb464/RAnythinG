Về chúng tôi

Local RAG cho tài liệu của bạn

Anything · powered by RAG — chạy trên máy bạn, không lock-in cloud.

Sứ mệnh

RAnythinG giúp bạn hỏi đáp và làm việc với tài liệu riêng — hợp đồng, báo cáo, spec kỹ thuật — mà không phải đưa dữ liệu lên dịch vụ bên thứ ba.

Chúng tôi tin rằng AI hữu ích nhất khi bám nguồn thật, chạy nơi bạn kiểm soát được, và minh bạch về cách trả lời.

Công nghệ

  • Backend: FastAPI, SQLAlchemy, FAISS
  • ML: sentence-transformers, transformers, PyTorch
  • Storage: Filesystem hoặc PostgreSQL + pgvector
  • Frontend: HTML/CSS/JS (app) + Jinja2 (marketing)
  • Deploy: Docker Compose

Roadmap

  • ✓ Core RAG: upload, chat, citations
  • ✓ Website sản phẩm đa trang
  • … Studio API trên UI chính
  • ○ Hỗ trợ model LLM tùy chọn (Ollama)
  • ○ Export notebook / backup
  • ○ Multi-user & auth
