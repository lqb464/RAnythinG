Lịch sử cập nhật

Theo dõi những thay đổi mới nhất của RAnythinG.

v1.2.1 2026-07-21 Mới

Graph mode (GraphRAG UI)

  • Toggle Assembly / Graph trên workspace; entity graph React Flow + communities
  • Build Graph on-demand (rule hoặc LLM); click entity → Ask
  • API /graph, /graph/build, /graph/ask — auto-index GraphRAG vẫn tắt mặc định
v1.2.0 2026-07-21 Mới

Assembly Canvas + landing gọn

  • Workspace React Flow: kéo nguồn, nối context vào Chat, gắn Studio artifact lên board
  • Quiz chấm điểm · Flashcard lật · Mind map expand
  • Landing tập trung sản phẩm; nav marketing thu gọn
v1.1.0 2026-07-21 Mới

Multi-user, LLM linh hoạt, export, service API có token

  • JWT đăng ký/đăng nhập; notebook theo owner_id
  • Khóa /api/external/* bằng EXTERNAL_API_TOKEN (Bearer từ service client)
  • Gemini multi-key + OpenAI-compatible / Ollama / vLLM (Settings UI)
  • Export / import notebook ZIP; sửa notebook id dài (user_*) + Studio persist trên Postgres
  • Tests + RAG golden smoke trong CI
v0.3.0 2026-07-08 Mới

Studio thật, Chat streaming, index tăng dần

  • Studio (Tóm tắt, Quiz, Flashcards, Mind Map, Report, Audio/Video Overview, Infographic, Slides, Data Table) sinh nội dung bằng LLM ngay trên UI chính, qua API /api/notebooks/{id}/studio/{tool}
  • Chat trả lời theo kiểu streaming (SSE) và nhớ vài lượt hội thoại gần nhất
  • Thêm/xóa tài liệu trong notebook đã index chạy tăng dần thay vì lập lại chỉ mục từ đầu
  • Gỡ bỏ Streamlit UI legacy — chỉ còn một giao diện HTML/CSS/JS duy nhất
