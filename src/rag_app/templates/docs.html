Từ zero đến notebook đầu tiên — chạy local hoặc Docker.

Cài đặt Docker Luồng sử dụng CLI Kiến trúc

Cài đặt (Python)

python -m venv .venv
.\.venv\Scripts\Activate.ps1   # Windows
pip install -r requirements.txt
python app.py

Mở http://localhost:8000 (trang sản phẩm) và http://localhost:8000/app (ứng dụng).

Docker (khuyên dùng)

docker compose up --build
  • App + PostgreSQL (pgvector) trong container
  • Postgres: port 5432, user/pass/db = ranything
  • FAISS index → volume app_data

Kiểm tra storage: GET /api/health

Luồng sử dụng

  1. Tạo notebook mới tại /app
  2. Upload tài liệu ở panel Sources (tự index)
  3. Chọn nguồn muốn dùng cho chat
  4. Hỏi trong Chat hoặc dùng câu hỏi gợi ý
  5. Dùng Studio để tạo quiz, flashcard, báo cáo…

CLI (tùy chọn)

python -m src.rag_app.cli build --source-folder ./docs --output-dir ./data
python -m src.rag_app.cli query --index-path ./data/rag_index.faiss \
  --metadata-path ./data/rag_index.faiss.json --query "Nội dung chính?"
