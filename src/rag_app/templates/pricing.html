Câu hỏi thường gặp

Có cần trả phí API OpenAI/Google không?

Không bắt buộc. RAnythinG dùng model local (E5 embedding + Qwen Instruct). Bạn có thể thay model sau nếu muốn.

Dữ liệu có lên cloud không?

Không, khi chạy local hoặc self-host. File và index nằm trên máy/volume của bạn.

Giới hạn số notebook / file?

Không giới hạn cứng — phụ thuộc dung lượng ổ đĩa và RAM khi index.
